JioFiDash

JioFiDash is a simple dashboard to manage your JioFi device.

Features

- Battery status, Network speeds, and Data usage.
- Change WiFi SSID name and password.
- View and Block connected WiFi devices.
- Enable the WPS push button.
- Restart JioFi device.
- Show notifications for low, full battery.
- Beautiful Dark material design theme.

Picar

A self driving car model that uses machine learning and image recognition to navigate a track.

GitHub

Trained an artificial neural network that predicts driving directions from the camera feed. Build the RC car with motor controllers, servos, ultrasonic sensors, camera and a Raspberry Pi. Server-client services that control movement and sensor data across the network. Haar cascade models to detect sign boards and traffic lights.

Raspberry Pi 3, Pi camera, Ultrasonic sensors, Motor controller(L298N), Servo and DC motors, 18650B cells, Step down 5V DC, Li-ion charging circuit
